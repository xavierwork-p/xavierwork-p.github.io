# Xavier Peralta - Portafolio

Sitio estático listo para GitHub Pages.

## Publicación recomendada

1. Crear el repositorio `xavierwork-p.github.io` en GitHub.
2. Subir el contenido de esta carpeta a la rama `main`.
3. GitHub Pages publicará el sitio en:

```text
https://xavierwork-p.github.io
```

No requiere servidor, base de datos ni pasos de compilación.
